# CryptoArena v1.2 — Release Notes

## Summary

v1.2 adds the full engineering backbone: smart contract testing pipeline (Hardhat),
production observability (Prometheus + Grafana), sentiment-aware regime detection,
and GitHub Actions CI for both Python and Solidity.

---

## New Modules

### 🔨 Hardhat Pipeline
- `hardhat.config.js` — Base L2 + Base Sepolia network config, BaseScan verification
- `package.json` — npm scripts: `compile`, `test:contracts`, `test:coverage`, `deploy:sepolia`
- `scripts/hardhat_deploy.js` — Ethers v6 deploy script with auto-save to `config/contracts.json`
- `scripts/hardhat_verify.js` — Auto-verify all contracts on BaseScan

### 🧪 Contract Tests (Hardhat / Chai)
- `test/KarmaToken.test.js` — 6 tests: name/symbol, mint, burn, access control, supply
- `test/ArenaLeaderboard.test.js` — 5 tests: CRUD, events, access control, dedup
- `test/AgentNFT.test.js` — 5 tests: mint, mapping, dedup guard, events

### 📊 Prometheus + Grafana
- `monitoring/prometheus.yml` — scrape config (arena_core metrics)
- `monitoring/docker-compose.yml` — Prometheus + Grafana containers
- `monitoring/grafana/` — auto-provisioned datasource + dashboard
- `monitoring/grafana/dashboards/arena_main.json` — 8-panel main dashboard
- `src/monitoring/metrics.py` — `ArenaMetrics` singleton (15 metrics: counters, gauges, histograms)

### 🌡️ Sentiment Engine
- `src/sentiment/sentiment_engine.py` — 4-source aggregator:
  - Fear & Greed Index (alternative.me, free)
  - CryptoPanic news (keyword scoring)
  - Twitter/X real-time search (Tweepy)
  - BTC funding rate proxy (CoinGecko derivatives)
- 5-minute cache, graceful fallback to 0.0 on any failure

### 🌊 Regime Detector v2
- `src/regime/detector_v2.py` — 6-signal composite detector:
  - Price action (SMA20/50 cross) — 30%
  - Sentiment — 25%
  - Volatility regime (Calm/Normal/Volatile/Explosive) — 20%
  - Volume relative — 10%
  - Funding rate pressure — 10%
  - Whale heuristic (exchange flows) — 5%
- Outputs `RegimeResult` with full signal breakdown
- Crisis detection: Explosive vol + negative composite → immediate Crisis

### ⚙️ GitHub Actions CI
- `.github/workflows/contracts_ci.yml` — Node 20, compile + test + coverage on PR
- `.github/workflows/python_ci.yml` — Python 3.11/3.12 matrix, lint + pytest on PR

## How to Run

```bash
# Smart contract workflow
npm install
npm run compile
npm run test:contracts
npm run deploy:sepolia

# Monitoring stack
docker-compose -f monitoring/docker-compose.yml up -d
# Grafana → http://localhost:3000  (admin / cryptoarena)
# Prometheus → http://localhost:9090

# Sentiment test
python -c "import asyncio; from src.sentiment.sentiment_engine import SentimentEngine; \
           s = SentimentEngine(); print(asyncio.run(s.get_sentiment()))"
```
