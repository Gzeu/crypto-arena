## 📜 Smart Contract: AgentMarketplace.sol

- **Trustless escrow**: AgentNFT locked on listing, returned on delist
- **KARMA payments** with automatic fee split (2.5% → treasury, rest → seller)
- **ReentrancyGuard + Ownable** (OpenZeppelin v5)
- **Events**: `AgentListed`, `AgentSold`, `AgentDelisted`, `FeeUpdated`, `TreasuryUpdated`
- **Configurable fee** up to max 10%
- **View helper** `getListing(tokenId)` for off-chain queries

## 🌐 REST API Endpoints (port 8001)

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/health` | Health check + version |
| GET | `/marketplace/listings` | Browse active listings (filter + sort) |
| POST | `/marketplace/listings` | Create new listing |
| POST | `/marketplace/listings/{id}/buy` | Purchase an agent |
| DELETE | `/marketplace/listings/{id}` | Delist own agent |
| GET | `/marketplace/price-suggest` | ML price recommendation |
| GET | `/marketplace/stats` | Volume, fees, totals |
| WS | `/marketplace/ws/live` | Real-time 10s snapshots |

---

## 🧠 Price Suggestion Model

Heuristic formula:
```
suggested_price = 100 (base)
                + win_rate × 500
                + max(total_pnl × 0.1, 0)
                + tournaments_won × 200
```

Example: `win_rate=0.65, pnl=1200, tournaments=2` → **1045 KARMA**

## 🚀 Quick Start

```bash

# Browse listings

curl http://localhost:8001/marketplace/stats

open http://localhost:8001/docs
```
